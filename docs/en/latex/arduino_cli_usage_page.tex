\begin{DoxyCode}{0}

These libraries will be installed in\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{\%USERPROFILE\%\(\backslash\)Documents\(\backslash\)Arduino\(\backslash\)libraries}

\end{DoxyCode}


To make Arduino CLI use them in a profile-\/based build (sketch.\+yaml)\+:
\begin{DoxyEnumerate}
\item First, run a compile command once, so Arduino CLI creates the internal library folders\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{arduino-\/cli\ compile\ YourProjectName\ -\/-\/clean\ -\/-\/log-\/level\ warn\ -\/-\/log\ -\/-\/build-\/path\ \textcolor{stringliteral}{"{}\%USERPROFILE\%\(\backslash\)Documents\(\backslash\)Arduino\(\backslash\)YourProjectName\(\backslash\)build"{}}}

\end{DoxyCode}

\item Then replace the {\ttfamily src/} folders in the following internal directories\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{\%USERPROFILE\%\(\backslash\)Documents\(\backslash\)Arduino\(\backslash\)platforms\(\backslash\)internal\(\backslash\)ESPAsyncWebServer...}
\DoxyCodeLine{\%USERPROFILE\%\(\backslash\)Documents\(\backslash\)Arduino\(\backslash\)platforms\(\backslash\)internal\(\backslash\)AsyncTCP...}

\end{DoxyCode}
 Overwrite the contents with the new files from the Git\+Hub libraries. 
\end{DoxyEnumerate}
