Microcontroller Specifications

Technical specs for the required ESP32-S3 module.

Required Microcontroller:

  • Board Order Code: ESP32-S3-WROOM-1-N16R8
  • ROM: 384 KB
  • SRAM: 512 KB
  • RTC SRAM: 16 KB
  • Flash Size: 16 MB (Quad SPI)
  • PSRAM Size: 8 MB (Octal SPI)
