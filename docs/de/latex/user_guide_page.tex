\chapter{User Guide}
\hypertarget{user_guide_page}{}\label{user_guide_page}\index{User Guide@{User Guide}}
Eine umfassende Anleitung zur Nutzung des Geräts.\hypertarget{user_guide_page_autotoc_md5}{}\doxysubsubsection{\texorpdfstring{Übersicht}{Übersicht}}\label{user_guide_page_autotoc_md5}
Dieses Gerät verbessert die Qualität deines 3D-\/\+Drucks durch eine optimale Lagerung des Filaments. Diese Anleitung zeigt Schritt für Schritt, wie du das System effektiv nutzt.\hypertarget{user_guide_page_autotoc_md6}{}\doxysubsubsection{\texorpdfstring{Erste Schritte}{Erste Schritte}}\label{user_guide_page_autotoc_md6}

\begin{DoxyEnumerate}
\item {\bfseries{Einschalten\+:}}
\begin{DoxyItemize}
\item Gerät mit Strom verbinden.
\item Beim Start werden Temperatur und Luftfeuchtigkeit auf dem Display angezeigt.
\end{DoxyItemize}
\item {\bfseries{Filament-\/\+Presets wählen\+:}}
\begin{DoxyItemize}
\item Mit dem Drehencoder durch voreingestellte Materialien (PLA, PETG, ABS usw.) blättern.
\item Gewünschtes Material per Knopfdruck auswählen. Das Gerät passt automatisch die Werte an.
\end{DoxyItemize}
\item {\bfseries{Manuelle Anpassung\+:}}
\begin{DoxyItemize}
\item Temperatur und Luftfeuchtigkeit lassen sich manuell über den Encoder einstellen.
\item Langes Drücken wechselt zwischen den Modi, Drehung passt die Werte an.
\end{DoxyItemize}
\item {\bfseries{Betrieb\+:}}
\begin{DoxyItemize}
\item "{}\+Start"{} aktiviert Heizung und Lüfter.
\item "{}\+Stop"{} beendet den Vorgang.
\end{DoxyItemize}
\end{DoxyEnumerate}\hypertarget{user_guide_page_autotoc_md7}{}\doxysubsubsection{\texorpdfstring{Anzeigen}{Anzeigen}}\label{user_guide_page_autotoc_md7}
LEDs zeigen den aktuellen Status\+:
\begin{DoxyItemize}
\item {\bfseries{Heizungs-\/\+LED}} – leuchtet bei aktiver Heizung
\item {\bfseries{Lüfter-\/\+LED}} – leuchtet bei laufendem Lüfter
\item {\bfseries{System-\/\+LED}} – zeigt den Gesamtstatus an
\end{DoxyItemize}\hypertarget{user_guide_page_autotoc_md8}{}\doxysubsubsection{\texorpdfstring{Wartung}{Wartung}}\label{user_guide_page_autotoc_md8}
Für dauerhaft gute Leistung\+:
\begin{DoxyItemize}
\item Heizung und Lüfter regelmäßig reinigen.
\item Steckverbindungen überprüfen.
\item Trocknungsmittel im Filamentbehälter bei Bedarf austauschen.
\end{DoxyItemize}\hypertarget{user_guide_page_autotoc_md9}{}\doxysubsubsection{\texorpdfstring{Fehlerbehebung}{Fehlerbehebung}}\label{user_guide_page_autotoc_md9}

\begin{DoxyItemize}
\item {\bfseries{Kein Strom\+:}} Stromquelle und Verbindung prüfen.
\item {\bfseries{Werte ändern sich nicht\+:}} Sensorverbindung kontrollieren.
\item {\bfseries{Keine LEDs\+:}} Verkabelung und Bauteile überprüfen.
\end{DoxyItemize}

Weitere Hilfe gibt es auf der Supportseite oder direkt beim Hersteller.
